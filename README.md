# CLASE2



- **algoritmo**: secuencia de instrucciones lógicas, ordenadas y finitas

**estructura**
- **input**: entrada  
- **proceso**: lista de instrucciones  
- **output**: resultado  

**diagrama de flujo**
- representación gráfica de un algoritmo  
- usa símbolos estándar  

**lenguaje de programación**
- entre 700–800 lenguajes  
- ejemplos: python, javascript  

---

**P5.JS**

**funciones**
- **setup()**: configura el entorno inicial  
- **draw()**: bucle infinito, genera movimiento e interacción  

**background(v1, v2, v3, [a])**
- define color de fondo  
- valores en rgb  

**dibujo**
- sistema de coordenadas  
- (0,0) esquina superior izquierda
